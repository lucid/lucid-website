Open Source Web Desktop

Lucid is a free, open source web based desktop, web desktop, or webOS that gives you a portable, online workspace that you can use to store files, play media, and manage your office documents. The goal of the project is to provide an application platform that integrates seamlessly with the web, existing desktop technologies, and mobile devices. Learn More→
